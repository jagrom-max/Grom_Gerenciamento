# Release Notes - v2.0.0

Data: 2026-05-07
Tag: v2.0.0

## Resumo
A versao 2.0.0 marca a migracao oficial do sistema Grom Gerenciamento da base legada em Python para a plataforma atual em PHP.

## Destaques da versao
- Migracao consolidada dos modulos principais para PHP.
- Evolucao dos fluxos de RH, escalas, operacional e relatorios.
- Atualizacao de controllers, rotas, views e modelos para padrao unificado da aplicacao web.
- Inclusao de suporte a PDF de escala mensal e ajustes de impressao.
- Expansao de migracoes e factories para dar suporte ao novo modelo de dados.
- Atualizacao da base de testes para refletir o comportamento da versao PHP.

## Impacto
- Esta release substitui oficialmente o fluxo principal em Python pelo fluxo em PHP.
- A tag v2.0.0 representa o marco de transicao da plataforma.

## Compatibilidade
- Recomendado executar migracoes de banco antes de uso em ambientes novos.
- Revisar variaveis de ambiente e dependencias conforme `README.md` e documentacao em `docs/`.
